# Podcast RSS Feed Generator (Python)

This repository contains a **Python-based RSS feed generator** that creates a podcast-compatible XML file from structured YAML data.

The project demonstrates how to convert human-readable metadata into a valid RSS 2.0 feed using Python, making it useful for learning backend automation, XML generation, and data transformation.
